준비 중

보려는 사이트에 현재 기본 페이지가 없습니다. 업그레이드 중이거나 구성 중일 수도 있습니다.

나중에 다시 시도하십시오. 계속 문제가 발생하면 웹 사이트 관리자에게 문의하십시오.

사용자가 웹 사이트 관리자이고 이 메시지가 잘못 보고된 것으로 생각되면 IIS 도움말의 '동적 콘텐트 사용 설정 및 해제'를 참조하십시오.

IIS 도움말을 보려면
  1. 시작을 클릭한 후 실행을 클릭하십시오.
  2. 열기 텍스트 상자에 inetmgr라고 입력하십시오. IIS 관리자가 표시됩니다.
  3. 도움말 메뉴에서 도움말 항목을 클릭하십시오.
  4. 인터넷 정보 서비스를 클릭하십시오.
